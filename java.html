Linguagem Java


A linguagem de programação serve para escrever uma série de comandos que podem ser usados para resolver problemas.


Java é uma linguagem multiplataforma, orientada a objetos e centrada em rede que pode ser usada como uma plataforma em si. É uma linguagem de programação rápida, segura e confiável para codificar tudo, desde aplicações móveis e software empresarial até aplicações de big data e tecnologias do servidor. A linguagem do java começa com uma public class, ou classe pública, por ser uma linguagem fortemente orientada a objeto.
